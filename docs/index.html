The bal website

bal is an extensible, commandline tool to manage personal finances and accounts. Its command prompt allows you to add transactions and keep track of your current and future account balances.

The program is highly extensible because the bal command prompt is actually a Scheme interpreter so it can execute arbitrary code and operate on transactions and accounts. bal also include facilities for making Scheme functions interactive using the Scheme function (bal/call) (which plays a similar role as (interactive) in Emacs, if you are familiar with it).

bal is Free Software licensed under the GNU General Public License Version 3. Its source code is available to download on Github.
